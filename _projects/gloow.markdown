	<h3>{{ page.title }}</h3>
	<br/>
	<b>Gloow</b> is an experimental arcade game built in Openframeworks. The game uses computer vision to detect two coloured paddles, which the player must use to steer a set of orbs and an interconnecting wire between obstacles.
	<br/>
	<br/>
	The game is built to allow for both single and multiplayer in the same mode. Multiplayer adds a dimension of communication that makes the game more enjoyable to watch and play.
