<b>Gloow</b> is a modern interpretation of classic 'wire-loop' games. Players interact with the game by moving two coloured paddles to steer a set of connected orbs around obstacles. This invisible link between player and screen adds a physical dimension to the traditional arcade game design.

The game is built to allow for both singleplayer and multiplayer, with multiplayer adding a dimension of communication that makes the game more enjoyable to both watch and play.

<b>Gloow</b> was built in <b>OpenFrameworks</b> using OpenCV for computer vision.
